# Wybe AI

**AI-kollega, ikke et verktøy.**

Wybe er en kollega du ansetter — svarer telefonen, følger opp leads, booker møter, sender epost, oppdaterer CRM. Proaktiv, multi-kanal, jobber 24/7.
